### 강조

```
_italic_ , *italic*  : 이탤릭(기울임)체
**hello** : bold(굵은 글씨)체
~~hello~~ : 취소선
```

_italic_, **hello**, ~~hello~~

<br>

### 코드

1. 앞에 4개의 공백

2. ` 3개 or ~3개로 감싸기

3. 인라인 - `로 감싸기

<br>

### 첨자

    - 윗첨자 : sup 태그 사용 <br>

예시 : a<sup>2</sup><br>

    - 아랫첨자 : sub 태그 사용 <br>
    예시 : 𝑙𝑜𝑔<sub>2</sub>𝑛<br>

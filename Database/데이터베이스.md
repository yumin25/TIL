SQL 문법 종류

- DDL(Data Definition Language)
  테이블과 컬럼 정의.
  - Create
  - Alter
  - Drop
  - rename
  - truncate
- DML(Data Manipulation Language) : 데이터 조작 언어
  데이터베이스의 내부 데이터 관리. 데이터 조회, 추가, 변경, 삭제
  - Select
  - Insert
  - Update
  - Delete
- DCL(Data Control Language) : 데이터 제어 언어
  - Commit
  - Rollback
  - Revoke : 사용자 권한 회수
  - Grant : 사용자 사용 권한 부여

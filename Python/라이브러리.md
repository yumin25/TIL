## random

```python
import random
```

* randint(a,b)<br>
a와 b 사이의 랜덤한 정수를 반환<br>
    ```python
    >>> random.randint(1,5)
    3
    ```

* sample(s,n)<br>
  s : 시퀀스 데이터 (range, list, tuple, string) 또는 set <br>
  n : 인자의 개수<br>
  s 중에서 n개의 랜덤하고 unique한 데이터를 순서와 상관 없이 뽑아 리스트로 반환
    ```python
    >>> data = random.sample(range(100), 10)
    >>> data
    [67, 26, 42, 71, 55, 18, 73, 78, 47, 59]
    ```

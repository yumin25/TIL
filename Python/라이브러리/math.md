```python
import math
```
<br>

* sqrt() : 제곱근
```python
>>> math.sqrt(9)
3.0
```

* pow(a,b) : a<sup>b</sup><br>
```python
>>> math.pow(2,4)
16.0
```

* ceil() : 올림
```python
>>> math.ceil(-1.111)
-1
```

* floor() : 내림
```python
>>> math.floor(-1.111)
-2
```

* fabs() : 절댓값 반환
```python
>>> math.fabs(-5)
5.0
```

* factorial() : 팩토리얼 함수
```python
>>> math.factorial(5)
120
```

## 집합 자료형 (set)

[점프투파이썬](https://wikidocs.net/1015)

* 중복 허용 x
* 순서 x 
<br>-> 인덱싱 불가. 리스트 or 튜플로 변환해야 한다.

### 관련 함수
* 값 1개 추가 : add
```python
>>> s1 = set([1,2,4])
>>> s1.add(3)
>>> s1
{1, 2, 3, 4}
```

* 값 여러 개 추가 : update
```python
>>> s1 = set([1,2,5])
>>> s1.update([3,4,5])
>>> s1
{1, 2, 3, 4, 5}
```

* 값 제거 : remove
```python
>>> s1 = set([1,2,3])
>>> s1.remove(2)
>>> s1
{1,3}
```

## 선택 정렬 (Selection Sort)
<br>
최솟값을 선택해 맨 앞부의 데이터와 교환해 정렬하는 알고리즘<br><br>

```python
def selection_sort(data):
    for i in range(len(data)-1):
        low = i
        
        for k in range(i+1, len(data)):
            if data[low] > data[k]:
                low = k

        data[i], data[low] = data[low], data[i]

    return data
```
<br>

시간 복잡도 : O(n<sup>2</sup>)

## 순열 (Permutation)

<sub>n</sub>P<sub>r</sub><br>
n개 중에서 r개를 뽑아 일렬로 나열하는 것.

<br>

특징

- <sub>n</sub>P<sub>r</sub> = n * (n-1) * .... *(n-r+1)
- <sub>n</sub>P<sub>n</sub> = n!

<br>

### 재귀
```python
def permutation(cnt):
    global count
    if cnt == r:
        print(number)
        count += 1
        return
    else:
        for i in range(0,n):
            if isSelected[i] == 1:
                continue
            number[cnt] = i
            isSelected[i] = 1
            permutation(cnt+1)
            isSelected[i] = 0

n = 5
r = 2
count = 0   #순열 개수 저장
number =  [ 0 for i in range(r)]    #순열 저장
isSelected =  [ 0 for i in range(n)]
permutation(0)

print(count)
```

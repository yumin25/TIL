## 배열 (Array)

선언
```java
int [] num = new int[4];
int [] num = {1,2,3,4};
```
<br>

* Arrays.toString() : 배열 값 출력. 일차원 배열 출력할 때 사용.
```java
import java.util.Arrays;

int[] num = {1,2,3,4};
System.out.println(Arrays.toString(num));

[1, 2, 3, 4]
```
<br>

* deepToString : 다차원 배열 출력.
```java
import java.util.Arrays;

int[][] num = {{1,2,3,4}, {5,6,7,8}};
System.out.println(Arrays.deepToString(num));

[[1, 2, 3, 4], [5, 6, 7, 8]]
```

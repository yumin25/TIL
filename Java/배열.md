## 배열 (Array)

선언
```java
int [] num = new int[4];
int [] num = {1,2,3,4};
```
<br>

* length : 배열의 길이
```java
int [] a = {1,2,3,4};
System.out.println(a.length);

4
```

<br>

*  clone() : 배열 내용 복사.

```java
b = a.clone();
```
